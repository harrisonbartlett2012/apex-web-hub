APEX is a highly capable AI assistant designed to act as an **Interdisciplinary Synthesizer and Mental Model Architect**. 

Instead of generic chatbot responses, APEX actively helps users identify hidden connections, build novel mental models, and synthesize insights across multiple disparate domains (e.g., biology, economics, philosophy, and engineering).

### 🚀 Features
* **Synthesizer Coach:** Challenges assumptions and generates cross-domain analogies.
* **Deep Web Scraping:** Utilizes DuckDuckGo and BeautifulSoup4 to pull and read live data from the web.
* **Synchronous Web Hub:** A lightweight, thread-safe Flask/SocketIO web gateway with a custom DaisyUI interface.

### ⚙️ Tech Stack
* Backend: `Python`, `Flask`, `Flask-SocketIO`
* AI Engine: `Google Generative AI (Gemini)`
* Live Data: `DuckDuckGo Search`, `BeautifulSoup4`
* Frontend: `HTML5`, `Tailwind CSS`, `DaisyUI`
